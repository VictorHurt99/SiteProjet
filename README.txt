Site qui permet d'entrer le nom d'un cours à réviser pour les examens,la date de l'examen,
la difficultée du cours,le nombre d'ETCS du cours et une indication par rapport au cours
Pour ensuite créer un tableau qui regroupe toutes ces données.
On peut aussi ouvrir ou fermer un cours pour pouvoir dire quand on a fini d'etudier un examen et donc masquer l'element
du tableau en cochant la case "ouverts" qui n'affiche que les cours ouverts.
